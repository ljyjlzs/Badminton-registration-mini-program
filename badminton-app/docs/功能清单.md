## 三、三种赛制完整流程对比

### 单打流程
```
报名(每人一队) → 小组赛(循环赛) → 挑战赛(末位 vs 首位) → 决赛(前2名) → 结束
```
- 队伍数：全程不变 = 报名人数
- 挑战赛：直接取最后一名
- 决赛：取前2名直接对阵

### 双打轮换流程
```
报名 → 分组(蛇形，C(n,2)种搭档) → 小组赛(循环赛) → 挑战赛(无共享成员匹配) → 决赛(个人前4蛇形组队) → 结束
```
- 队伍数：分组时产生多个临时队伍 → 决赛新建2个队伍
- 挑战赛：从末位扫描无共享成员
- 决赛：个人积分前4，1+4 vs 2+3，新建 `is_final_team` 队伍

### 双打固搭流程
```
报名(选搭档) → 分组(2人一队) → 小组赛(循环赛) → 挑战赛(末位 vs 首位) → 决赛(前2名) → 结束
```
- 队伍数：全程不变 = 报名人数/2
- 挑战赛：直接取最后一名
- 决赛：取前2名直接对阵，**不新建队伍**
